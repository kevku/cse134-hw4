Modern Algebra (Math 103 A~B)

Groups

Groups:

A Group is a set bounded by a binary operation usually denoted as (G, *) where G is a set and * is the binary operation. For it to be a group, it must satsify these properties:

  1. Closure
  2. Associativity
  3. Idenitity Element
  4. Inverse Element
Closure: Arbitrary elements a, b, ∈ G, such that a * b = c, c needs to be an element in G.
Associativity: Arbitrary elements a, b, c, ∈ G, then (a * b) * c = a * (b * c)
Identity Element In group G, there must exist an element e such that arbitrary element g ∈ G satistfies g * e = g.
Inverse Element For all elements g ∈ G, there exist a g-1 such that g * g-1 = e.
(Abelian Groups) Commutativity Arbitrary elements a, b, ∈ G, a * b = b * a.
Isomorphism

Isomorphism:

2 Groups are considered Isomorphic when there exist a mapping Φ such that:

  1. Φ(a) * Φ(b) = Φ(a * b)
  2. There is a bijection in Φ
